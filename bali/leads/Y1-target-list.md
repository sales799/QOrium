### Specific targeting categories (the agent prospects from)

**Tier-A — Talpro Universe alumni network** (~50 firms, 1st-degree warm intros)

- Filled by Bali (CEO Y1) — LinkedIn export + filter, the AI Agent doesn't pull these.
- Highest conversion probability; Y1 ICP for the first 10 logos.

**Tier-C — LinkedIn Sales Navigator open-search** (~500 firms)

- AI Agent runs daily, capped at 50 prospects/day per the system prompt + escalation rubric.

### Conversion pacing

Y1 funnel math (per Bali Playbook §11 conversion benchmarks for Recruiter):

```
LinkedIn outbound prospects/day (cap)        =  50
Connection accept rate                        ~  35%        ≈ 18 connects/day
DM reply rate                                 ~  10%        ≈ 1.8 replies/day
Trial signup rate                             ~  30%        ≈ 0.5 trials/day
Trial-to-paid conversion                      ~  25%        ≈ 0.13 paid/day
                                                              ≈ 3-4 logos/month
                                                              ≈ 35-45 logos/year
```

That's slightly over the 30-logo Y1 target — accommodates churn + ramp. Y1 expectation: agent online M2 → first 3 months are ramp + tuning → real conversion volume from M5 onward.
