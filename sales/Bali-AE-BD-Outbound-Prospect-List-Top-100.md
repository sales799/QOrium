## §1 — Account selection criteria

Each account scored on 5 dimensions per Bali Playbook §3.4:
1. **Hiring volume** (technical roles/year)
2. **Stack alignment** (does QOrium Wave 1+2 content cover their stack?)
3. **Procurement complexity** (4-9 months Indian GCC norm; vs platforms 3-6)
4. **Decision-maker accessibility** (warm intro paths; existing Talpro India relationships)
5. **Anchor revenue value** (₹40L Stack-Vault Enterprise vs $25K ReadyBank API)

Threshold: composite score ≥ 14/25 to make the Top 100.

## §7 — Outreach motion per tier

### Tier 1 motion (Stack-Vault Enterprise/Group):

1. CTO Office prepares pre-call brief per account (using E2-style stack research)
2. CEO sends warm-intro email/WhatsApp via existing relationship OR LinkedIn InMail
3. AE Enterprise (when hired) takes the discovery call
4. Sample 50-Q pack delivered within 2 weeks of discovery call
5. Multi-stack scoping conversation (4-9 month procurement)
6. MSA + DPA + Order Form signing

### Tier 3 motion (ReadyBank API platforms):

1. BD Platforms outreach (when hired) at VP Product / Head of Content
2. API documentation hand-off + sandbox access
3. Pilot integration scoped (4 weeks)
4. Pilot live for 8 weeks
5. Commercial negotiation
6. Long-form agreement signing

### Tier 4 motion (JD-Forge Enterprise/Reviewed):

1. Landing page conversion (paid + organic)
2. Self-service signup → first JD-Forge order
3. CSM follow-up after first order if positive signal
4. Account growth via 50+ JDs/month → enterprise tier upsell
