## How to read this doc

Each sprint has 5 fields:

1. **Description (3-5 sentences):** What's happening in plain English
2. **Status:** ✅ DONE · 🟢 ACTIVE · 🟡 NEXT · 🔵 FUTURE · 🔴 BLOCKED
3. **Done-when:** Concrete checklist (3-7 items) that defines "complete"
4. **Unlock-cascade:** What becomes possible when this sprint completes
5. **Notes/links:** References to specific docs
