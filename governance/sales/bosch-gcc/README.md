## Anti-patterns to avoid

- Selling on the discovery call. **Discovery is asking, not telling.**
- Quoting hard prices. **Range-only per BRAND.md.**
- Promising features beyond shipped capability. **No "we'll have
  X next quarter" if X requires cred-drop or human-bound items.**
- Disrespecting Bosch's audit/compliance posture. **Bosch GCC has a
  high bar; understand it before pitching against it.**
- Over-emphasising India-stack. **Bosch GCC's primary lens is
  global; India-stack is secondary.**
